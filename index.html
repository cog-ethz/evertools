EVE R Tools

An R package to evaluate data gathered with the EVE framework.

View on GitHub Download .zip Download .tar.gz

Welcome to EVE R Tools.

The evertools-package is meant to be used in tandem with the soon to be released EVE framework. The package's purpose is to wrap access to our database into simpler R-calls and provide some basic statistical analysis over data points in the database.

You can download the source code here or use R's devtools to get evertools into your R-distribution and work with it.

Installation

To install evertools, you need to have devtools installed in your R-distribution. Then the command

devtools::install_github("cog-ethz/evertools")

will download an install the package for you and you can access it by adding

library(evertools)

to you R-script.

Usage

The package is mainly developed to expose statistical evaluation to our Unity-based experiment framework EVE. While we use RDotNet to load these R-scripts into the C-sharp environment and will make mostly use of features in the package, we will allow others to load their own R-scripts to enable more complex evaluations within our framework. Or if you prefer to work in the R-environment, you can use evertools to access the database and then proceed in R.
